## Overview

This repository contains a collection of Python-based projects and exercises demonstrating practical applications of data handling, statistical analysis, and machine learning. The projects cover a wide range of topics including data manipulation, visualization, regression, classification, clustering, anomaly detection, and neural networks.  

Each project includes Python code, datasets, and outputs to illustrate real-world applications of Python and Data Science concepts.

---

## Features & Topics

- **Data Handling & Analysis:**  
  Using `pandas` and `numpy` for cleaning, filtering, and aggregating datasets.

- **Data Visualization:**  
  Creating plots using `matplotlib` and `seaborn`.

- **Statistical Analysis:**  
  Hypothesis testing, ANOVA, regression analysis.

- **Machine Learning Algorithms:**  
  - Logistic Regression  
  - Decision Trees & Random Forest  
  - K-Nearest Neighbors (KNN)  
  - Support Vector Machines (SVM)  
  - Naive Bayes Classifier  
  - Principal Component Analysis (PCA)  
  - K-Means & Hierarchical Clustering  
  - Anomaly Detection  

- **Neural Networks & Deep Learning:**  
  - Basic Neural Networks  
  - Convolutional Neural Networks (CNNs)  

- **Natural Language Processing (NLP) Basics:**  
  Text vectorization and sentiment analysis using Python.
